# Projeto Ovitrampa

## Descrição
Esse projeto esta sendo realizado para acompanhar os casos de dengue na cidade de Brasilândia MS.
O projeto consiste em criar um mapa interativo para acompanhar os casos de dengue e as informações das armadilhas "Ovitrampas"
